# Health Fabrics Website

قالب بسيط لعرض محتوى حول الأقمشة والملابس الصحية. يعتمد التصميم على الخطوط والصور المستخدمة في الموقع الأصلي مع بعض التعديلات الإبداعية. الصفحة الرئيسية مصممة لاستقبال نصوص وصور مخصصة بسهولة.

## طريقة الاستخدام

1. ضع الملفات في مسار سيرفر الويب الخاص بك.
2. عدل النصوص داخل `index.html` لتناسب محتواك.
3. توجد صورة الخلفية في المسار `images/hero-background.png` ويمكن استبدالها بما
   يناسبك مع تعديل المسار في ملف `style.css` إذا لزم الأمر.
4. عدل روابط التنقل والعناوين بما يتناسب مع محتوى موقعك.
